# Fundamentos de Ciencias 2026

Material del curso **Fundamentos de Ciencias** (Ingeniería en Salud / Biomédica).

## Contenido

### Clases
- **Clase 1**: La Química de la Vida
- **Clase 2**: Dogma Central de la Biología Molecular, Regulación Genética, Epigenética y ARN reguladores

### Archivos incluidos
- Transcripciones automáticas de las clases (usando Whisper)
- Apuntes de cada clase
- Diapositivas de las clases (PDF)
- Artículos científicos relevantes:
  - CRISPR-Cas9 y sistemas de edición genética
  - Epigenética y regulación transgeneracional
  - Factores de transcripción
  - Regulación a nivel de traducción

### Script de transcripción
`transcribe.py` - Script en Python que utiliza el modelo `whisper-medium` para transcribir audios de clases en español y generar archivos Markdown bien formateados.

## Tecnologías
- Python
- OpenAI Whisper (medium model)
- CUDA (para aceleración GPU)
